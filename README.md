# Smart Study Session Tracker

## Problem
Students often struggle with tracking their actual study time and staying consistent.

## Solution
This project allows users to:
- Start and stop study sessions
- Track time spent on each subject
- View study summary

## Features
- Simple CLI-based system
- Offline functionality
- Tracks real-time sessions

## How to Run
1. Clone the repository
2. Run:
   python main.py

## Example
Start session → Study → Stop → View summary

## Learning Outcomes
- Time tracking using Python
- JSON data handling
- Real-world problem solving
